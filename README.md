# WIHN Landing Page (2026 Brand-Aligned)

Responsive React + Tailwind + Framer Motion landing page for **Women in Hospitality Nigeria (WIHN)**.

## Run

```bash
npm install
npm run dev
```

Build:

```bash
npm run build
```

## Brand Compliance Notes

- Primary CMYK `(71, 100, 33, 41)` converted to RGB `44, 0, 101` (`#2C0065`) and nearest web-safe `#330066`.
- Logo lockup is centered in hero and implemented with protected clear-space (`1em`, approximating one "M" width).
- Minimum logo symbol render size enforced at `83x60px` (22x16mm equivalent).
- Secondary accents include orange, pink, green, turquoise blue, purple, and dark orange.
